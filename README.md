NAME
====

P5wantarray - Implement Perl 5's wantarray()/scalar() built-ins

SYNOPSIS
========

    use P5wantarray;

DESCRIPTION
===========

This module tries to mimic the behaviour of `wantarray` and `scalar` of Perl 5 as closely as possible.
